# 📋 Student Report Card

A Python program that generates a student report card.
Calculates total, average, grade and pass/fail result.

---

## 🛠️ Tech Stack
- Python

## 📊 Features
- Stores student data using Dictionary
- Calculates total marks using Loop
- Calculates average automatically
- Assigns Grade (A/B/C/F) based on average
- Displays clean formatted report card

## 📸 Output
```
===== REPORT CARD =====
Name   : Shanti
Marks  : [78, 85, 90, 60, 72]
Total  : 385
Average: 77.0
Grade  : B
Result : Pass
=======================
```

## 🚀 How to Run
```bash
python report_card.py
```

- Variables & Data Types
- Functions & Return
- Loops & Conditions
- Lists & Dictionaries
